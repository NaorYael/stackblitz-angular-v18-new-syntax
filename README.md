## Introduction

This is a user and post management application built with Angular. It fetches user data from an API and displays it alongside the associated posts. The application showcases the use of Observables and reactive programming with RxJS, enhancing user experience through smooth data handling.

## Features

- Fetches and displays a list of users.
- Displays posts associated with each user upon clicking their name.
- Utilizes Angular's standalone components for better organization.
- Implements a responsive and user-friendly design.
- Error handling for API calls.

## Technologies Used

- **Angular**: A platform for building mobile and desktop web applications.
- **RxJS**: A library for reactive programming using Observables.
- **TypeScript**: A typed superset of JavaScript that compiles to plain JavaScript.
- **HTML5** & **CSS3**: For structuring and styling the application.
